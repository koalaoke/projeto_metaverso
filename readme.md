# Projeto: Meu Primeiro Ambiente VR
Este projeto consiste em uma experiência de Realidade Virtual (VR) desenvolvida 
na Unity utilizando o Meta XR SDK. O ambiente simula um espaço interativo 
com foco em física e manipulação de objetos esportivos e decorativos.

O projeto foi estruturado para ser leve, organizado e funcional no Meta Quest (Android), 
respeitando as boas práticas de hierarquia e nomenclatura.

## Rodando 
1. Requisitos:
   - Unity 6+.
   - Módulo Android Build Support instalado.

2. Instalação:
   - Clone este repositório.
   - Abra a pasta raiz no Unity Hub.
   - Aguarde a importação dos pacotes.

3. Execução:
   - Navegue até a pasta: Assets/Cenas/
   - Abra a cena: "CenaPrincipal".
   - Para testar no PC: Dê Play no Editor (movimentação simulada).
   - Para VR: Vá em File > Build Settings, selecione Android e clique em Build and Run 
     com o Quest conectado.
